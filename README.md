# Ontico Kanban

Канбан-доска для заявок конференции [ontico.ru](https://ontico.ru).

## Требования

Для работы расширения необходимо быть авторизованным на сайте [conf.ontico.ru](https://conf.ontico.ru). Расширение использует данные вашей учётной записи для загрузки заявок. Без авторизации данные не будут доступны.

## Установка в Chrome

### Вариант 1: из ZIP-архива (готовая сборка)

1. Скачайте файл `ontico-kanban.zip` из [релизов](../../releases).
2. Распакуйте архив в любую папку на вашем компьютере.
3. Откройте Chrome и перейдите по адресу `chrome://extensions/`.
4. Включите **Режим разработчика** (переключатель в правом верхнем углу).
5. Нажмите кнопку **«Загрузить распакованное»**.
6. Выберите папку, в которую вы распаковали архив.
7. Расширение появится в списке и будет готово к использованию.

### Вариант 2: сборка из исходников

1. Убедитесь, что установлены [Node.js](https://nodejs.org/) (v18+) и npm.
2. Клонируйте репозиторий:
   ```bash
   git clone <url-репозитория>
   cd ontico
   ```
3. Установите зависимости:
   ```bash
   npm install
   ```
4. Соберите расширение:
   ```bash
   npm run build
   ```
5. Откройте Chrome и перейдите по адресу `chrome://extensions/`.
6. Включите **Режим разработчика** (переключатель в правом верхнем углу).
7. Нажмите кнопку **«Загрузить распакованное»**.
8. Выберите папку `dist` внутри проекта.

## Использование

1. Войдите в аккаунт на [conf.ontico.ru](https://conf.ontico.ru).
2. Нажмите на иконку расширения **Ontico Kanban** в панели инструментов Chrome.
3. Расширение загрузит ваши заявки и отобразит их в виде канбан-доски.
