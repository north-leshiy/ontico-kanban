### Requirement: Панель фильтрации по кураторам
В хедере доски SHALL отображаться панель с аватарами кураторов, реально присутствующих хотя бы на одной карточке текущего набора заявок. Кураторы без ни одной назначенной заявки SHALL быть исключены из панели.

#### Scenario: Панель отображает только активных кураторов
- **WHEN** заявки загружены
- **THEN** в панели фильтра показываются только кураторы, чьи ID встречаются в `curator.value` хотя бы одной заявки

#### Scenario: Панель пуста
- **WHEN** ни одна заявка не имеет назначенного куратора
- **THEN** панель фильтра не отображается

### Requirement: Мультиселект кураторов
Пользователь SHALL иметь возможность выбрать одного или нескольких кураторов кликом по аватару. Повторный клик по выбранному куратору SHALL снимать выбор. Логика фильтрации — OR: заявка отображается если назначен хотя бы один из выбранных кураторов.

#### Scenario: Выбор куратора
- **WHEN** пользователь кликает по аватару куратора в панели фильтра
- **THEN** аватар выделяется визуально (ring/border), доска скрывает карточки без этого куратора

#### Scenario: Выбор нескольких кураторов
- **WHEN** пользователь выбрал двух и более кураторов
- **THEN** на доске показываются карточки, у которых в `curator.value` есть хотя бы один из выбранных

#### Scenario: Снятие выбора куратора
- **WHEN** пользователь кликает по уже выбранному аватару
- **THEN** куратор снимается с выбора; если выбранных не осталось — показываются все карточки

### Requirement: Сброс фильтра
Рядом с аватарами SHALL присутствовать кнопка "Все" (или "×"), которая активна только при наличии выбранных кураторов. Клик по ней SHALL сбрасывать все выборы.

#### Scenario: Сброс через кнопку
- **WHEN** пользователь нажимает кнопку "Все" при активном фильтре
- **THEN** все выборы снимаются, на доске показываются все карточки

#### Scenario: Кнопка неактивна без фильтра
- **WHEN** ни один куратор не выбран
- **THEN** кнопка "Все" отображается в неактивном/деактивированном виде или скрыта

### Requirement: Tooltip с именем куратора в фильтре
При наведении на аватар куратора в панели фильтра SHALL отображаться tooltip с полным именем куратора.

#### Scenario: Tooltip на аватаре в фильтре
- **WHEN** пользователь наводит мышь на аватар в панели фильтра
- **THEN** показывается tooltip с `curator.name`
